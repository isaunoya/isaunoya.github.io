新高考一卷

7.

$c = \ln \frac{10}{9} < \frac{1}{9}$ 因此 $b > c$.
$a = 0.1 e^{0.1}$,众所周知 $x + 1 \leq e^x \leq \frac{1}{1-x}$

$\Delta: e^x = \sum_{}^{\infin} \frac{x^n}{n!}, \frac{1}{1-x} = \sum_{}^{\infin} x^n[0 \leq x < 1]$

所以 $a < 0.1\ \frac{10}{9} = \frac{1}{9}$, 所以 $b > a$.

重点在 a 和 c 的比较上

显然可以构造函数, $\phi(x) = x\ e^x + \ln (1-x)$

$\phi’(x) = (x+1) e^x - \frac{1}{1-x}$

$\frac{(x+1)\ e^x}{\frac{1}{1-x}} = (1-x^2)\ e^x$ ，所以 $\varphi(x) = (1-x^2)e^x, \varphi’(x) = (1-2x - x^2)e^x \geq 0$ 在 $[0, 0.2]$ 成立，因而 $\varphi(x) \geq \varphi(0) = 1$，所以 $\phi’(x) \geq 0, \phi(x) \uparrow$.

$\phi(0.1) > \phi(0) = 0, a>c$.

所以答案是 $C.$

21.

首先 $\frac{2^2}{a^2} - \frac{1^2}{a^2-1}=1, a^2=2, C: \frac{x^2}{2}-y^2=1$.

$l:y=kx+b$, $x^2 - 2(kx+b)^2 - 2 = 0, (1-2k^2)x-4kbx-2b^2-2=0$

$\frac{y1-1}{x1-2} + \frac{y2-1}{x2-2} = 0 \rightarrow (y1-1)(x2-2) + (y2-1)(x1-2) = 0$

$(kx1+b-1)(x2-2)+(kx2+b-1)(x1-2)=0 \rightarrow 2kx1x2 + (b-1-2k)(x1+x2)-4(b-1)=0$

$x1 + x2 = \frac{4kb}{1-2k^2},x1x2=\frac{-2b^2-2}{1-2k^2}$

$\rightarrow \frac{-4k(b^2+1)+(b-1-2k)4kb}{1-2k^2} = 4(b-1)$

$\rightarrow 4kb^2-4k+8bk^2+4kb-4kb^2 = 8bk^2-4b-8k^2+4$

$\rightarrow 4kb+4k+4b-4+8k^2=0$

$\rightarrow 2k^2 + k + kb + b - 1 = 0$

$\rightarrow (k+1)(2k + b - 1) = 0$

$\rightarrow k = -1$

$l:y=-x+b\ [k = -1]$

$\rightarrow \tan (\alpha - \beta) = 2\sqrt 2$

$\rightarrow \frac{k1-k2}{1+k1k2} = 2\sqrt 2$

因为 $k1 + k2 = 0$

解出来 $k1=\sqrt 2,k2 = -\sqrt 2$ 或者 $k1 = \frac{\sqrt{2}}{2}, k2 = -\frac{\sqrt 2}{2}$

然后就是 DIRTY WORK, 最后答案 $\frac{16\sqrt 2}{9}$
