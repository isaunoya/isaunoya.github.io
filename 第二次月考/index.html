7.

statement:

第一排是 1,3,5,7,…,2n-1

每个数都是肩上两数之和。（第二排是 4,8,12,16,20…）

然后对于一个数，我们可以把它划分成一个倒三角形（n 行）。

solution:

显然倒三角形的第一行每个数 $a_i$ 贡献了 $\binom{n}{i}$（下标从 0 开始)。

所以三角形最底下的那个数字就是 $\sum \binom{n}{i} a_i$。

然后题目问的是 64 行第 17 个数字。

把三角形找出来，然后第一排是 33…159。

那么我们只需要求 $S = 33 \binom{64}{0} + 35 \binom{64}{1} + … + 159 \binom{64}{64}$

同时 $S = 159\binom{64}{0} + 157\binom{64}{1} + … + 33\binom{64}{64}$

所以 $2S = 192(\sum \binom{64}{i}) = 192 \times 2^{64}$

$S = 3 \times 2^{68}$

8.

statement:

$f(x) = x^2 - x \ln (ax) - \frac{e^{x-1}}{a}$，如果 $(0, \infin)$ 有零点，a 的取值范围。

solution:

$f(x) = x(x - \ln (ax) - \frac{e^{x-1}}{ax})$

因为 $x > 0$，所以零点等价于 $g(x) = x - \ln(ax) - \frac{e^{x-1}}{ax}$ 的零点。

$g(x) = 0 \leftrightarrow x - 1 - \ln(ax) - \frac{e^{x - 1}}{ax} = -1$

$\because x - 1 - \ln(ax) = \ln \frac{e^{x-1}}{ax}$

令 $t = \frac{e^{x-1}}{ax}$

题目即要求 $\ln t - t = -1$。

显然仅有 $t = 1$ 时取到。

即要求 $\frac{e^{x-1}}{ax} = 1 \leftrightarrow a = \frac{e^{x-1}}{x}$

对 $\frac{e^{x-1}}{x}$ 求导得到取值范围 $a \in [1, \infin)$。

21.

statement:

$\frac{x^2}{4} + \frac{y^2}{3} = 1$

$S_{ABQ} = \frac{1}{2}$, $|F_1 A| = \lambda|F_2B|$,求$\lambda + \frac{1}{\lambda}$

solution:

显然 $S_{F_1F_2Q} = S_{ABQ}, S_{F_1AF_2} = \lambda \times \frac{1}{2}, S_{F_1BF_2}=\frac{1}{2\lambda}$

所以 $S_{F_1F_2AB} = \frac{1}{2}(\lambda + \frac{1}{\lambda} + 2)$

$F_1A = \frac{3}{2-\cos \theta}, F_2B = \frac{3}{2+\cos\theta}$

$S_{F_1F_2AB} = F_1A \times F_2B \times 2 \times \sin\theta = \frac{12 \sin \theta}{4-\cos^2\theta}$

所以 $\frac{24 \sin \theta}{4-\cos^2\theta} = (\lambda+1) + \frac{\lambda + 1}{\lambda}$

$\large \lambda + 1 = 2 S_{F_1F_2A}= \frac{6\sin\theta}{2-\cos \theta} \rightarrow\frac{\lambda + 1}{\lambda} = \frac{\lambda + 1 +\frac{\lambda + 1}{\lambda}}{\lambda + 1} = \frac{\frac{24 \sin \theta}{4-\cos^2\theta}}{\frac{6\sin\theta}{2-\cos \theta}} = \frac{4}{2+\cos\theta}$

$\frac{\lambda + 1}{\lambda} = \frac{6\sin\theta}{2+\cos\theta}$，解得 $\sin \theta = \frac{2}{3}, \cos \theta = \frac{\sqrt 5}{3}$

所以 $\large \lambda = \frac{F_1A}{F_2B} = \frac{\frac{3}{2-\cos \theta}}{\frac{3}{2+\cos\theta}} = \frac{2+\cos\theta}{2-\cos\theta} = \frac{6+\sqrt 5}{6 - \sqrt 5}$

$\lambda + \frac{1}{\lambda} = \frac{(6+\sqrt 5)^2 + {(6 - \sqrt 5)^2}}{(6+\sqrt 5)(6 - \sqrt 5)} = \frac{82}{31}$
