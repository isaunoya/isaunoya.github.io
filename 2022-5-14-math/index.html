8.

$f(x) = \frac{e^x +e^{-x}}{2}, g(x) = \frac{e^x - e^{-x}}{2}, mf(x)g(x) - e[mf(x) + g(x)] + e^2 \leq 0$ 在 $[-1, 1]$ 恒成立，$m$ 取值范围.

A. $(-\infin, \frac{2e^2}{e^2 +1}]$ B. $(-\infin, e]$ C. $[\frac{2e^2}{e^2 +1}, \infin)$ D.$[e, \infin)$

desmos 得到答案 D.

因式分解: $(mf(x) - e)(g(x) - e) \leq 0$

$g(x) - e \leq 0$ 所以只需要 $m f(x) \geq e, m \geq \frac{e}{f(x)}$
得到 D.

12.

$a_n$ 都是正数，$S_n = \frac{a_n}{2} +\frac{1}{2a_n}$。

A.$S_n^2$ 等差 B. $S_n +S_{n+2} < 2S_{n+1}$ C. $a_{n+1}>a_n$ D. $S_n - \frac{1}{S_n} \geq \ln n$

$S_n^2 = \frac{a_n^2}{4} + \frac{1}{4an^2} + \frac{1}2$, $S_{n-1}^2 = (S_n-a_n)^2 = \frac{a_n^2}{4} + \frac{1}{4a_n^2}-\frac{1}2$

$S_n^2 - S_{n-1}^2 = 1$，成立，A 正确。

$a_1 = \frac{a_1}{2} + \frac{1}{2a_1}, a_1 > 0$,因此 $a_1 = 1$, $S_n^2 = n$

$S_n + S_{n+2} < 2S_{n+1}$，$\sqrt{n} +\sqrt{n+2} < 2\sqrt{n+1}$,$f(x) = \sqrt x$ 是个凸函数，所以显然成立，B 正确。

$a_n = f(n) = \sqrt{n} - \sqrt{n-1}$。, 求导显然 $f’(x) < 0$，所以 $a_{n+1} < a_n$，C 错误。

D 即证明 $\sqrt{n} - \frac{1}{\sqrt{n}} \geq \ln n$。

$f(x) = (x)^{1/2} - (x)^{-1/2} - \ln x$

$f(1) = 0$, $f’(x) = \frac{1}2x^{-1/2} + \frac{1}2x^{-3/2} - \frac{1}{x}$

$\frac{1}2x^{-1/2} + \frac{1}2x^{-3/2} \geq 2 \sqrt{\frac{x^{-2}}{4}} = x^{-1}$

$f’(x) \geq 0, f(x) \geq 0$，得证，D 正确。

20.

$E:y^2 = 4x, H(3,4)$

过点 H 作两条直线 $l1, l2$, 与 $E$ 交于 $A, B, C, D$，$AB, CD, AD, BC$ 的斜率分别为 $k1, k2, k3, k4$。$AB, CD$ 的中点连线得到的斜率是 $k$。

求证: $\frac{1}{k1} + \frac{1}{k2} = \frac{1}{k3} + \frac{1}{k4}$ 且 $\frac{1}{k3} + \frac{1}{k4} - \frac{1}{k}$ 为定值。

设 $A(4a^2, 4a), B(4b^2, 4b), C(4c^2, 4c), D(4d^2, 4d)$。

得到:$AB: x = (a+b)y - 4ab, CD: x = (c+d)y - 4cd, AD: x = (a+d)y - 4ad, BC: x = (b+c)y - 4bc$

第一问证明: $\frac{1}{k1} = a+b, \frac{1}{k2} = c+d, \frac{1}{k3} = a+d, \frac{1}{k4} = b+c$。

显然得证。

然后 $AB, CD$ 过 $H$。

因此 $3 = 4a +4b - 4ab = 4c +4d - 4cd$。

$AB$ 中点 $E(4(\frac{a^2+b^2}{2}), 4\frac{a+b}{2})$， $CD$ 中点 $F(4(\frac{c^2+d^2}{2}), 4\frac{c+d}{2})$。

$2(a^2+b^2) = 2((a+b)^2-2ab) = 2(a+b^2)+3-4a-4b$

因此 $E(2(a+b)^2+3 - 4a-4b, 2(a+b)), F(2(c+d)^2 +3-4c-4d,2(c+d))$

$EF: x=(a+b+c+d-2)y-4\frac{a+b}{2}\frac{c+d}{2}+3$

$\frac{1}{k} = a+b+c+d - 2$.

$\because \frac{1}{k3}+\frac{1}{k4} = a+b+c+d$

定值为 2.

21.

存在 $x \geq 1,$ $f(x) = \ln x +ax^2 - x\geq \frac{1}{2} x^3 + 1$

移项得到 $a \geq g(x) = \frac{\frac{1}{2} x^3 + x + 1 - \ln x}{x^2}$

$g’(x) = \frac{1}{2} - \frac{1}{x^2} + \frac{2\ln x - 3}{x^3} = \frac{\frac{1}{2}x^3 - x + 2 \ln x - 3}{x ^ 3}$

$h(x) = \frac{1}{2}x^3 - x + 2\ln x - 3$

$h’(x) = \frac{3}{2} x^2 - 1 + \frac{2}{x} = \frac{3x^3 - 2x +4}{2x}$

$h’(x) > 0$ 恒成立。（求 $\Delta$）

$h(\frac{3}{2}) = -\frac{45}{16} + 2 \ln \frac{3}{2} < 0$

$h(2) = 2 \ln 2 - 1 > 0$

$x0 \in (\frac{3}{2}, 2)$

$g(x)\min = g(x0) = \frac{\frac{1}{2} x0^3 + x0 + 1 - \ln x0}{x0^2}$

$\because \frac{1}{2}x0^3 - x0 + 2 \ln x0 - 3 = 0$

$-\ln x0 = \frac{1}{4}x0^3 - \frac{1}{2}x0 - \frac{3}{2}$

得到 $g(x0) = \frac{\frac{3}{4}x0^3 + \frac{1}{2} x0 - \frac{1}{2}}{x0^2} = \frac{3x0^3 +2x0 - 2}{4x0^2}$

$\phi(t) = \frac{3}{4}t - \frac{1}{2t^2} -\frac{1}{2t^2}$

$\phi’(t) = \frac{3}{4} - \frac{1}{2t^2} + \frac{1}{t^3} = \frac{3t^3 - 2t + 4}{4t^3}$

怎么分子在上面出现了一次

所以 $\phi’(t) > 0$

$\because x0 \in (\frac{3}{2}, 2)$

$\phi(\frac{3}{2}) < \phi(x0) < \phi(2)$

$\phi(\frac{3}{2}) = \frac{89}{72}, \phi(2) = \frac{13}{8}$

$a \geq \phi(x0), a \in Z, a_{\min} = 2$
