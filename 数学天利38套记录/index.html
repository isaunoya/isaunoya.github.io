Posted on 2022-06-02

1.浙江省金华十校高三期末调研考试:

8.statement:

已知数列 {$a_n$} 有 $a_1=1,a_2=2$,满足 $\frac{a_{n+2}}{a_n} = \frac{a_{n+1}^2}{a_n^2+1}$,估计 $a_{2022}$ 的范围:

solution:

$\frac{a_{n+2}}{a_n} = \frac{a_{n+1}^2}{a_n^2+1} \rightarrow \frac{a_{n+1}^2+1}{a_{n+2}} = \frac{a_n^2+1}{a_n} \rightarrow \frac{a_{n+1}^2 + 1}{a_{n+2}a_{n+1}} = \frac{a_{n}^2+1}{a_na_{n+1}}=\frac{a_1^2+1}{a_1a_2} = 1 \rightarrow a_{n+1}=a_n+\frac{1}{a_n}$

$a_{n+1}^2 = (a_n+\frac{1}{a_n})^2 > a_n^2 + 2 \rightarrow a_{n} \geq \sqrt{2n-1}$

$a_n^2 = 2n-1 + \frac{1}{a_1^2} + \frac{1}{a_2^2} … \leq 2n-1 + (n-1) = 3n-2$

$\sqrt {2n-1} \leq a_n \leq \sqrt{3n-2}$

16.statement:

$e1 \cdot e2 = -\frac{31}{32}, 0 < e3\cdot e4\leq \frac{\sqrt 3}{2}$, $\min |e1 + e2 + e3 + te4|$

solution:

$|e1+e2|^2 = \frac{1}{16}, |e1+e2| = \frac{1}{4}, |e3+te4| \geq \frac{1}{2}$

$|e1+e2+e3+te4| \geq |e3+te4| - |e1+e2| = \frac{1}{4}$
